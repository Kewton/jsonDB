# 概要
- ローカルにjsonDB

# インストール方法
```
pip install git+https://github.com/Kewton/jsonDB
```

# 開発環境の準備
```
python -m venv .venv
.venv\Scripts\activate.ps1
.venv\Scripts\python.exe -m pip install --upgrade pip
pip install -r requirements.txt
pip install pytest
pip install mkdocs
```
